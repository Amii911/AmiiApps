# AMIIAPPS

AmiiApps is a web application designed to help software engineering interview candidates track their technical interview preparation in a structured and organized way.

Instead of using spreadsheets or generic tools, AmiiApps provides a centralized dashboard to manage problems solved across platforms like LeetCode, NeetCode, HackerRank, and CodeSignal.
---
## Purpose

AmiiApps replaces manual spreadsheet tracking with a purpose-built system designed specifically for technical interview preparation.


## MVP Features

- User registration and secure authentication

- Create, edit, and delete problem entries

- Track problem title, platform, difficulty, and status

- Add optional notes and links for review

- Centralized dashboard to view all tracked problems

- Filter problems by difficulty, platform, and status
